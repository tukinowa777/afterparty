# 二次会居酒屋ファインダー

夜9時以降に入れる二次会向け居酒屋を、現在地と条件から3件提案する Web アプリです。

## 使い方

1. `python3 server.py --port 8123` を実行する
2. PC では `http://localhost:8123` を開く
3. iPhone / Android では、同じ Wi-Fi 上でこのPCのLAN IPに `:8123` を付けて開く
4. 必要なら「現在地を使う」で位置情報を許可する
5. 人数、予算、料理ジャンル、移動距離を指定する
6. 条件に合う候補を最大3件確認する

例: `http://192.168.1.10:8123`

## 補足

スマホで現在地取得を安定させるなら HTTPS 配信が望ましいです。LAN 内の簡易確認では、ブラウザやOSの設定次第で位置情報許可が必要になります。

## 現在の開発状況

- フロント画面と条件検索は実装済み
- 現在地取得による再計算に対応済み
- 店舗データは `data/venues.json` に分離済み
- `server.py` から `GET /api/venues` で店舗一覧を返す簡易 API を追加済み
- まだ実店舗 API、DB、認証、地図連携は未実装

## 構成

- `index.html`: 画面構造
- `styles.css`: レイアウトと見た目
- `app.js`: API 読み込み、現在地取得、絞り込み、描画
- `server.py`: 静的配信と簡易 API
- `data/venues.json`: 店舗データ

現在はローカルのサンプル店舗データで動作します。実運用に寄せる場合は、店舗 API と接続して営業時間や価格帯を実データに差し替えてください。
